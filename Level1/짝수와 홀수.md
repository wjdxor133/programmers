## 내가 짠 코드

```jsx
function solution(num) {
  return num % 2 === 0 ? "Even" : "Odd";
}
```

## 문제 풀이

- 이 문제는 간단하다.
- 단순히 해당 수가 `짝수이면 "Even"`을 반환하고 `홀수이면 "Odd"`를 반환하는 문제이다.
- 해당 수를 2로 나눠서 나머지가 0이면 짝수이므로 `"Even"`을 반환
- 나머지가 0이 아니면 홀수이기 때문에 `"Odd"`를 반환한다.
